# **App Name**: Backend Mentor Evolved

## Core Features:

- Interactive Challenge Catalog: Browse and filter coding challenges by difficulty, tech stack, and tags.  Includes AI-powered recommendations based on skill level and learning goals.
- Live Code Editor: Write and execute code directly in the browser with support for multiple languages. Integrated with AI-powered code completion and error detection tool.
- AI Assistant (Rahim): Get real-time coding help, debugging assistance, challenge explanations, and personalized learning recommendations from an AI mentor tool. Rahim adapts to the user's coding style and provides tailored feedback.
- AI Code Review & Optimization: Receive automated feedback on code submissions, including performance, readability, best practices, and security vulnerabilities, from Rahim the AI Reviewer.  Suggests optimized solutions and alternative coding approaches tool.
- Personalized Learning Paths: AI-driven creation of custom learning paths based on user skill level, goals, and preferred technologies.  Includes recommended challenges, tutorials, and resources.
- Collaborative Coding Environments: Real-time collaborative coding environments where multiple users can work together on challenges. Integrated with voice and video chat.
- Gamified Learning: Earn points, badges, and rewards for completing challenges and contributing to the community. Leaderboards track progress and encourage friendly competition.
- Firebase Authentication: Secure user authentication using Firebase Auth with options for Google, email/password, and more.
- Real-Time Sync: Sync submissions, leaderboards, and other data in real-time using Firebase Firestore.
- AI-Powered Project Generation: Generate project ideas based on user skill level and chosen technologies. This tool will include detailed specifications, user stories, and architecture diagrams to jumpstart the development process.
- AI-Driven Tech Stack Recommendation: Based on the project the user is about to begin, this tool suggests which technologies to use. The LLM reasons about the problem, and then returns an answer.

## Style Guidelines:

- Primary color: A deep, calming blue (#4682B4) to evoke trust and competence.
- Background color: Light gray (#F0F8FF) for a clean and professional look.
- Accent color: A vibrant orange (#FF7F50) for call-to-action buttons and highlights, adding a touch of energy.
- Body and headline font: 'Inter' for a modern and readable sans-serif.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean and modern icons to represent different features and functionalities.
- Subtle transitions and animations to create a smooth and engaging user experience.
